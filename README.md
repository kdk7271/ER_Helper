## 🧐프로젝트 개요

* BERT 언어 모델을 응급상황을 5단계의 중증도로 분류하도록 파인튜닝하였다.
* 분류된 중증도에 따라 사용자 근처의 병원을 안내하는 서비스 포탈을 구현하였다.
